Телеграм-бот проводит игру, выбирает из списка (порядка 3000) вопросов 
и задает их пользователю. Вопросы с вариантами ответов, пользователю предоставляется 
клавиатура с кнопками для выбора варианта отвта и кнопкой "Завершить". В конце игры
выводится счет и предлагается ввести команду для начала новой игры.

Вопросы получены с сайта https://baza-otvetov.ru/categories/view/1

Для старта бота нужно запустить файл bot.py с помощью интерпретатора Python
и подождать порядка минуты (пока распарсится сайт с вопросами)
